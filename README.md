# ThingsLink
开源的iot平台

## 功能

- 设备连接到thingsLink并发送数据，连接协议包括MQTT,HTTP
- thingsLink 发送数据到设备,目前支持自定义数据
- 设备协议适配，对于原始的二进制报文，提供扩展插件进行内容转换
- 在线调试与数据报文查看
- 数据规则处理，提供消息转发与告警规则
- 数据统计分析
- 设备的websocket 消息

## 文档

## 快速开始

1. 定义产品，产品中约束设备的协议与payload 数据格式
2. 创建设备，自动分发设备连接方式所需的密钥等信息
3. 设备连接，[参考连接文章](things-link-docs/transport.md)
4. 在线调试与数据转发
